SatisfactoryDedicatedServer one-click start (and server protection) code version 1.0
This program comes with monitoring port 7777 to realize the automatic restart function when the server crashes

Put the SatisfactoryDedicatedServer folder in the C drive directory
First start Game download update file.bat
After the update is completed, there will be an additional GameServers folder in the folder
Finally, start start.bat
That's it
Game server update, please manually close all CMD pop-ups
